## 프로젝트 개요

**모두의집**은 오늘의집의 스토어와 커뮤니티 서비스를 클론코딩하는 프로젝트입니다!   
따라서 스토어(마켓) 팀과 커뮤니티 팀으로 나눠 API를 구현합니다.   
두 서비스에서 공통 사용하는 도메인(ex. `User`)이 존재하기 때문에 **멀티 모듈** 프로젝트를 도입하였습니다.

<br>
